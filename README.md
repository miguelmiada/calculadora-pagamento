#  Calculadora de Formas de Pagamento

Programa desenvolvido em **Linguagem C** que calcula o valor final de um produto com base na forma de pagamento escolhida pelo usuário.

---

##  Funcionalidades

O sistema permite:

* Inserir o valor de um produto
* Escolher a forma de pagamento
* Calcular automaticamente descontos ou juros

---

## 💳 Formas de Pagamento

| Opção | Forma de Pagamento           | Condição           |
| ----- | ---------------------------- | ------------------ |
| 1     | À vista em dinheiro          |  10% de desconto |
| 2     | À vista no cartão de crédito |  5% de desconto  |
| 3     | Em duas vezes                | Preço normal    |
| 4     | Em três vezes                |  10% de juros    |

---

## 🖥️ Como Compilar e Executar

### ⚙️ Compilar

```bash
gcc 1.c -o pagamento
```

### Executar

```bash
./pagamento
```
